# Material Design Component Library POC

Testing **material-leptos** (Material Design) with Leptos 0.6.

## Build

```bash
cd examples/material-poc
wasm-pack build --target web --dev
```

## Components Used

- `<MdButton>` - Material Design buttons
- `<MdCard>` - Material Design cards
